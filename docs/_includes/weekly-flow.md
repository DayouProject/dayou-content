## 每周工作总览

`周一 提交选题` → `周二 跑脚本并初审` → `周三 内容复核定稿` → `周四 生成视频` → `周五至周日 自动发布与复盘`

- 周一：内容负责人整理选题，大师/顾问补充核心概念，统一提交到 GitHub Issue。
- 周二：创始人根据本周选题跑内容管线，内容负责人开始看脚本初稿。
- 周三：大师/顾问重点检查文化准确性，内容负责人汇总反馈并确认定稿。
- 周四：创始人生成视频并检查发布队列是否正常。
- 周五至周日：系统按排期发布，内容负责人看账号表现并记录数据，周日报周报。

> 紧急问题走微信；可追踪、可回看的协作一律优先放在 GitHub Issue。
