# Dayou Avatar Script Template

Use this structure for every script. Keep the spoken copy tight enough for a 35-60 second vertical video.

## HOOK

One sentence. Start with tension, contradiction, or a pattern the viewer recognizes immediately.

## BODY

Three short beats maximum:

1. Explain the ancient line in plain English.
2. Connect it to a modern emotional or practical situation.
3. Land one memorable takeaway.

## CTA

One sentence. Invite reflection, comments, or follows. Keep it soft.

## CAPTION

One short paragraph plus hashtags.

Required output metadata:

- `title`
- `hook`
- `body_points`
- `cta`
- `caption`
- `hashtags`
- `visual_direction`
- `duration_seconds`
